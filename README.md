# Buek Core

Build AI Workers for Any Industry

Buek Core is a modular AI platform that separates AI reasoning from industry knowledge.

Instead of rebuilding AI for every industry, developers build once and extend the platform using
reusable domain modules.

## Vision

One AI Core.

Unlimited Industry Knowledge.

## First Vertical

Manufacturing

## Tech Stack

- OpenAI
- Codex
- React
- Express
- PostgreSQL
- Prisma
- Docker

## Getting Started

```bash
pnpm install
cp .env.example .env
pnpm db:generate
pnpm dev
```
