# ASM-022: Engine Bolt Torque Standard

## Scope

Applies to engine assembly station EA-04 and final torque confirmation for bracket and housing bolts.

## Symptoms

- Bolt torque out of specification.
- Repeated re-torque required.
- Assembly station shows abnormal torque trace.

## Possible Cause

- Worn torque tool calibration.
- Incorrect bolt lubrication.
- Cross-threading.
- Wrong tightening sequence.
- Operator skipped confirmation step.

## Inspection

1. Hold affected unit and identify station, operator, torque tool ID, and lot number.
2. Compare torque trace against standard curve.
3. Verify tool calibration date and fixture condition.
4. Inspect bolt thread, surface lubrication, and mating component.
5. Check last five similar cases in assembly history.

## Countermeasure

1. Recalibrate torque tool if drift is detected.
2. Replace damaged bolt and inspect mating thread.
3. Repeat tightening using approved sequence.
4. Run three consecutive OK torque traces.
5. Escalate to engineering if repeated on same station.
