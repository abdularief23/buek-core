# Troubleshooting: White Streak Defect

## Trigger

Use this guide when the operator reports "white streak", "missing line", "blank band", or "pale line" on printer output.

## Diagnostic Path

1. Confirm if the streak is vertical, horizontal, random, or repeated at fixed intervals.
2. Print a nozzle or density test pattern.
3. If the missing area appears on the test pattern, inspect nozzle, printhead, ink flow, and encoder strip.
4. If the streak repeats at a fixed interval, inspect transfer roller or feed roller.
5. If the defect appears only on one media batch, inspect dust, coating, humidity, and media storage.

## Root Cause Hypotheses

- Nozzle clog blocks ink flow.
- Toner or ink is below process minimum.
- Encoder strip contamination causes positioning error.
- Transfer roller has toner buildup, scratch, or flat spot.
- Preventive maintenance was skipped before production start.

## Suggested Countermeasure

- Run approved cleaning cycle.
- Clean encoder strip and media path.
- Replace consumable or transfer roller if cleaning fails.
- Add first article print evidence to setup checklist.

## Verification

Release only after three consecutive OK samples pass QC Standard 12.
