# ESC-002: Operator Escalation Rule

## Escalation Criteria

Escalate from operator to quality engineer when any of the following are true:

- The same defect appears in two consecutive inspection checks.
- Barcode, safety label, or traceability information is affected.
- First countermeasure fails.
- The defect appears after consumable replacement.
- Suspected material batch issue affects more than one machine.

## Required Evidence

- Defect photo.
- Machine ID.
- Work order or lot number.
- Material batch.
- Operator name.
- Timestamp and shift.
- Latest test pattern.

## Escalation Response

1. Quality engineer reviews sample and SOP evidence.
2. Production lead confirms containment scope.
3. Maintenance checks machine condition.
4. Release is allowed only after QC approves verification sample.
